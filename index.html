About Me

Hello, I'm Cory Carmen, a student software developer based on Vancouver Island and currently building my skills in full-stack web development.

After spending 25 years in the commercial diving and fishing industry, I made the decision to transition into tech and pursue software development full-time.

I enjoy building practical, user-focused projects while continuing to grow in JavaScript, React, Node.js, Express.js, MongoDB, and authentication systems.

From Commercial Diving to Software Development

My path into software development is not a traditional one. Before entering tech, I spent 25 years in the commercial diving and fishing industry, where safety, precision, teamwork, and problem-solving were part of everyday life.

Working in those environments taught me how to adapt quickly, stay calm under pressure, and keep learning through hands-on experience.

That background now influences how I approach development. I bring the same discipline and persistence to debugging, building projects, and learning new technologies that I brought to my previous career. Moving into software development has given me the opportunity to apply those strengths in a new field, and I am focused on continuing to grow as a full-stack developer.

My Journey

My development journey began when I enrolled in a Software Development Bootcamp and committed to building a strong foundation in modern web development.

Since then, I have been steadily expanding from core frontend skills into full-stack development through practical projects and hands-on learning.

Here is a snapshot of that progression:

  • Built a solid foundation in HTML5, CSS3, and modern web design fundamentals.
  • Developed version control skills with Git and GitHub as part of an organized development workflow.
  • Completed my first professional-style website project using semantic HTML and responsive design principles.
  • Strengthened frontend development skills with Bootstrap and mobile-friendly layout design.
  • Learned core JavaScript concepts and applied them to real interactive features.
  • Built dynamic user experiences through DOM manipulation, event handling, and modern ES6+ syntax.
  • Used Async/Await and the Fetch API to work with asynchronous data and external services.
  • Created professional contact form features with validation and a better user experience.
  • Integrated REST APIs using live weather data from OpenWeatherMap.
  • Used Postman for API testing, request building, and documentation practice.
  • Expanded into backend development with Node.js and Express.js.
  • Strengthened full-stack knowledge through the MERN stack and database work with MongoDB.
  • Started building frontend and mobile skills with React and React Native.
  • Began learning authentication workflows with Auth0 and Passport Local Strategy.
